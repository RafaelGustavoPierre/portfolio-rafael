Projeto em destaque

Vendas de Bebidas - Brewer

O Brewer é uma plataforma web para gerenciamento e comercialização de bebidas, desenvolvida utilizando Spring Boot, Thymeleaf, Bootstrap, JavaScript e MySQL. O projeto combina uma interface responsiva com funcionalidades robustas para cadastro de produtos, controle de estoque e automação de notificações por e-mail, demonstrando a integração entre front-end e back-end em uma aplicação completa.

Tecnologias HTML, CSS, JavaScript, Bootstrap, MySQL, Thymeleaf, Spring Boot
